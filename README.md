# NVC_train

[こちら](https://github.com/NON906/NVC)で使用する音源ファイル(*.nvz)を作成するためのツールです。

## Google Colaboratoryで実行（推奨）

Google Colaboratoryでnvc_train.ipynbを開き、その手順に従ってください。
生成したファイルの一部はGoogleドライブ上に保存されるため、あらかじめ空きを作っておいてください。

Google Colaboratory（[公式サイト](https://colab.research.google.com/)）はブラウザ上から無料で使える仮想環境です。
ブラウザが使えるPCであれば、OSを問わず使うことができます。
クラウド上に環境が作られるため、インストールなどによってローカルの環境が変わってしまうことはありません。
GPUを使用することもできるので、この方法を推奨します。

## Windows（ローカル）で実行

ローカル環境のWindowsで実行する場合は、nvc_train.mdを確認してください。
なお、NVIDIA製のGPUがあり、なおかつCUDA・cuDNNがインストールされている必要があります。
